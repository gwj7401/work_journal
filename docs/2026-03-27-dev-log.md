## OCR 引擎升级（2026-03-27 下午）
- **问题发现**：通过原文与 EasyOCR 识别结果截图对比，发现 EasyOCR 对表格文档存在严重缺陷：
  - 表格多列内容无间隔粘连（如`DATEDESCRIPTIONREFNOCHARGESCREDITBALANCE`变成一长串）
  - 左右两栏内容错行混排
  - 中文印章识别准确率极低（"广州机场空港希尔顿花园酒店" → "厅州场港斋尔顿.园。店"）
- **升级方案**：放弃 EasyOCR，改用 **PaddleOCR PP-StructureV2**（完全免费、本地运行）。
  1. 安装 `paddlepaddle` + `paddleocr` 依赖（已成功，exit code 0）。
  2. 完整重写 `scripts/pdf2word_ocr.py`：
     - 引入 `PPStructure` 版面分析引擎，自动区分标题/正文/表格/图片区域。
     - 表格区域提取 HTML 并解析为真实 Word 表格（`TableParser` + `python-docx` 表格API）。
     - 文本区域按纵向坐标排序，逐行写入，避免多列混排。
     - 图片/印章区域插入 `[图片区域]` 占位符。
     - 渲染分辨率从 2.0×提升至 2.5×（等效 300 DPI），进一步提升识别精度。
